Every time I build a static page I think, "wow, I should automate this." So while I work through the prework, I may take a few extra steps such as...
  • Install ruby 2.1.5
  • Install the Middleman gem
    • Oh, I guess I'll need bundler?
  • Templatize the page
  • Put content into yaml/markdown
  • Build it locally
  • Store the code in GitHub
  • Copy the built site to a server somewhere (I use Digital Ocean, but maybe BitBalloon will be easier since I won't have to mess with httpd config...).
